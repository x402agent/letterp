### P-Launch (Bonding Curves)

Pump-style constant-**P**roduct bonding curves ‚ÄĒ written in **P**inocchio, launched in seconds:

```
k = virtual_token_reserves √ó virtual_sol_reserves

Buy:   tokens_out = token_reserves ‚ąí (k / (sol_reserves + sol_in))
Sell:  sol_out    = sol_reserves   ‚ąí (k / (token_reserves + tokens_in))

Initial reserves:  30 SOL virtual ¬∑ 793.1B tokens virtual
Fee:               1% (100 bps) ‚Üí creator PDA
Graduation:        85 SOL raised ‚Üí auto-migrates to Raydium CPMM
```

### P-erps

Once a **P**-token graduates from the bonding curve, the **P**erps adapter lists it as a market on Drift or Adrena ‚ÄĒ giving holders leveraged exposure without leaving the **P** ecosystem.

### **P**ay.sh

**P**rivate x402 **P**ayment facilitation. Every launchpad API call can return `402 Payment Required`. Agents **P**ay in USDC or their own token. **P**ay.sh blinds the payer from the resource server:

```
Agent ‚Üí pay.sh relay ‚Üí resource endpoint
          ‚Üē blind signature
    Solana p-token USDC settlement (105 CU)
```
