# 08 — Serialization

Encode and decode account data and instruction arguments for SPL Token
and Token-2022 operations.

## Sub-modules
- `borsh_encode/` — Borsh serialization for instruction payloads
- `borsh_decode/` — Borsh deserialization for account data
- `pack_unpack/` — SPL Token Pack/Unpack trait implementations
