# 14 — Tests

Test suites for all pToken SDK modules. Covers unit tests, full-stack
integration tests, and fast local validator testing with Bankrun.

## Sub-modules
- `unit/` — Pure Rust unit tests for math, serialization, and validation
- `integration/` — Full program tests against a local Solana test validator
- `bankrun/` — Fast lightweight tests using solana-bankrun (no validator needed)
