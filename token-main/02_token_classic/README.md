# 02 — SPL Token Classic

Complete operation set for the original SPL Token program.
Program ID: `TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA`

## Sub-modules
- `mint_creation/` — Initialize new token mints
- `token_account/` — Create and initialize token accounts
- `transfer/` — Transfer tokens between accounts
- `approve_delegate/` — Approve a delegate for spending
- `revoke/` — Revoke delegate authority
- `burn/` — Burn tokens from supply
- `freeze_thaw/` — Freeze and thaw token accounts
- `close_account/` — Close token accounts and reclaim rent
- `multisig/` — M-of-N multisig authority setup
