# Program IDs

This directory records public program IDs for LetterP deployments.

Devnet IDs are generated locally and listed in `devnet/programs.toml`. The matching private keypairs live under `devnet/keypairs/`, which is intentionally ignored by git so this repository can be open sourced without publishing deploy authorities.

No mainnet program has been deployed from this workspace. Mainnet IDs should be generated from the release build artifacts and deployment authority process for the final audited programs.
